# Ticket Generator Service

Service de génération de tickets pour Event Planner - Création de QR codes et gestion anti-fraude.

## Fonctionnalités

- Génération QR codes
- Tickets sécurisés
- Signatures digitales
- Anti-fraude
- Templates personnalisables
- Export PDF

## Stack Technique

- Node.js + Express
- QR Code Generator
- PDF Generation
- PostgreSQL
- Redis (cache)

## Installation

```bash
npm install
npm start
```

## Développement

```bash
npm run dev
```

## Docker

```bash
docker-compose up -d
```
